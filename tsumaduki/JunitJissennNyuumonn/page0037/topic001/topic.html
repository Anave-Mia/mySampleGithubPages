起きたこと

「データクラス」と書かれている。

問題点

「データクラス」が未説明ワードで、周辺の文章の理解ができない。

原因

本書籍内の説明箇所への注を書いていない。

解決策

下記により、「データを保持するだけのクラスのこと」として読み進める。

「データを保持するだけのクラスのこと」を検索したが、確証度高い定義が見当たらない。
サイト(https://zenn.dev/t_fukuyama/articles/5b4fe1c72b7db4)にて
下のコードのような「データを保持するだけのクラスのこと」と書いてあった。 


                public class ContractAmount {
                    public int amountIncludingTax;  // 税込み金額
                    public BigDecimal salesTaxRate; // 消費税率
                }
